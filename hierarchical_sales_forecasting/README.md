# Hierarchical B2B Sales Forecasting

An open-source Python framework designed for Enterprise RevOps and Data Strategy teams. This library reconciles machine learning forecasting baselines with hierarchical quota target cascading, human managerial overrides (commits), and "sandbagging/happy ears" pipeline confidence algorithms.

Unlike traditional bottom-up time-series libraries (which are optimized for B2C retail/inventory forecasting), this package is explicitly designed for the complexities of enterprise B2B sales management.

### Features
* **Quota Cascading:** Distribute targets up and down complex organizational DAGs with built-in managerial hedges.
* **Commit Reconciliation:** Fuse algorithmic pipeline closing probabilities with judgemental (human) forecasts.
* **Managerial Bias Adjusted Forecasting:** Dynamically re-weight pipeline likelihood based on historical accuracy.

*(Library heavily inspired by scalable Databricks/Anaplan data engineering patterns.)*
